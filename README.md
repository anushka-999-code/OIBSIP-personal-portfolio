# personal-portfolio
task 2-Oasis Infobytes: Personal Portfolio

This repository contains my **Personal Portfolio** website, showcasing my projects, skills, and contact information. It is designed to highlight my abilities as a 2nd-year Computer Science student and my interest in web development.

**Features**
- Home section with introduction
- About Me section
- Skills section categorized by technical and soft skills
- Projects section showcasing mini projects
- Contact section with email, LinkedIn, and resume links
- Responsive design for mobile and desktop

**Technologies Used**
- HTML5, CSS3, JavaScript
- Git & GitHub for version control
- Optional: Figma or Canva for UI/UX design assets

  **How to Run**

1. Clone the Repository

git clone https://github.com/anushka-999-code/portfolio.git

2. Navigate to the Project Folder

3. Open the Portfolio in a Web Browser

4. Optional: Use Live Server (VS Code)
